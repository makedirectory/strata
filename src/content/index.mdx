# Strata

A **design-first, local-first, MCP-native** tool for modeling AWS infrastructure
as a typed, visual graph — and importing existing CloudFormation / Terraform to
see it rendered.

This documentation has two public sections:

- **[User Guide](/docs/guide)** — for everyone using the app: building diagrams,
  importing IaC, validation, saving/loading.
- **[Architecture & Engineering](/docs/architecture)** — for contributors and
  engineers: the service registry, domain model, persistence, MCP/IaC import,
  and the UX direction.

> Open source. Contributions welcome.
